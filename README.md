# aide-mq

Utility objects to ease usage of Bunny gem within Ruby applications

## Installation -- Not ready as not released yet
Add this line to your application's Gemfile:

```ruby
gem 'aide-mq'
```

And then execute:

```
$ bundle
```

Or install it yourself as:

```
$ gem install aide-mq
```
